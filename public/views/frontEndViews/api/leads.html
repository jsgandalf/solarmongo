Leads

Don't forget to pass in an access_token with every request

  • get: /leads/alldata - gets all leads with siteSurvey
  • get: /leads - gets all leads
  • post /leads - adds a lead, pass in the lead object with a siteSurvey Object
  • get /leads/:leadId - gets 1 lead, pass in the leadId, returns a lead object
  • put /leads/:leadId - updates a lead, pass in the leadId and lead object
